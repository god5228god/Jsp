○ 물리적 파일 구성
	- ResisterForm.jsp
	·회원 가입 폼 구성
	  (아이디, 이름, 이메일)
	·액션 태그를 활용하여 회원 가입 페이지 정체성 구성(include)
	-  Header.jsp
	· 각 페이지의 정체성을 전달하는 페이지로 구성
	   (액션 태그 활용)
	- RegisterProcess.jsp
	·회원 가입 폼으로부터 넘겨받은 데이터를 활용하여
	  객체를 생성하고(세션 스코프, 액션 태그 활용)
	·결과를 처리하는 페이지로 이 내용 전달
	  (액션 태그 활용)
	- RegisterResult.jsp
	·결과 출력
	·액션 태그 활용하여 페이지 정체성 구성(Header 사용)
	- com.model.UserBean
	·자바 빈 구성
