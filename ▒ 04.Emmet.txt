■■■ Emmet(에밋) ■■■

○ 개요

  - 기존에는 Zen Coding(젠코딩)이라는 이름으로 사용되었다.
  - 이후 Google에서 이 도구를 수용하면서 emmet이라는 이름으로 변경되었다.
  - HTML 생산성 향상 도구(HTML 코딩을 빠르게 수행하기 위한 플러그인)
  - CSS 선택자 사용(CSS3 지원)
  → HTML을 CSS 선택자를 통해 검색하는 방법 대신
     CSS 선택자를 통해 HTML을 생성하도록 하는 개념

○ 환경 설정

  - 현재(2026년 기준) 대부분의 툴에서 지원하고 있음
    (※ 그리고 대부분의 편집기 내에서의 액션 키 Ctrl+E)

○ 특이사항
  
  - Emmet 플러그인이 설치 이후 기존(이클립스) 단축키와 기능이 중복되는 항목이나
    관련 현상이 발생할 수 있음(Ctrl + D)
    → 단축키 재설정 후 사용(이클립스 > Window > Preferences > General > Keys ...)
